========================================================================
MacOS X PS Printer Driver
========================================================================


Introduction

This readme file contains information about using Postscript printer driver
for Mac OS X. Please make sure to read this before installing Postscript
printer driver.

<Contents>
  [1] Installing Postscript printer driver
  [2] Restriction and Notification in using this PPD File

========================================================================

  [1] Installing Postscript printer driver

    -1 To install the downloaded driver,
       open the downloaded .dmg file and double-click the install package.
    -2 To install the driver from the CD,
       open the driver folder and double-click the install package.
    -3 Install the driver following the installer guidance.
       The installer may ask you to enter the Administrator password or
       agree to the Software License Agreement.

  [2] Restriction and Notification in using this PPD File
    * Additional Page Count in Duplex Printing
      An extra page is added to the end of odd-paged duplex jobs.
      The added page might increment the printer's counter.
